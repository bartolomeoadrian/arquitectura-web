## Descripción

Este sistema se encargará de la compra y venta de autos así como Kavak

## Endpoints

| METHOD		| PATH						|
| ------------- | -------------------------	|
| GET			| /shops					|
| POST			| /shops					|
| GET       	| /shops/{id}				|
| PUT			| /shops/{id}				|
| DELETE		| /shops/{id}				|
| GET       	| /shops/{id}/cars			|
| POST			| /shops/{id}/cars			|
| POST			| /shops/{id}/cars/{id}/buy	|
| GET       	| /shops/{id}/cars/{id}		|
| PUT			| /shops/{id}/cars/{id}		|
| DELETE		| /shops/{id}/cars/{id}		|
| GET			| /users					|
| POST			| /users					|
| GET       	| /users/{id}				|
| PUT			| /users/{id}				|
| DELETE		| /users/{id}				|

## Instalar

```bash
$ npm install
```

## Iniciar

```bash
$ npm run start:prod
```
